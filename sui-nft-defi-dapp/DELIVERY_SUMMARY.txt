================================================================================
                            QUICK START GUIDE
================================================================================

1. BUILD MOVE CONTRACTS
   cd contracts/nft && sui move build
   cd ../defi_example && sui move build

2. PUBLISH TO DEVNET
   sui client publish --gas-budget 100000
   (Save the Package IDs from output)

3. UPDATE PACKAGE ADDRESSES
   Edit frontend/src/lib/suiClient.ts
   Replace: 0xNFT_PACKAGE and 0xDEFI_PACKAGE

4. INSTALL FRONTEND
   cd frontend && npm install

5. RUN DEV SERVER
   npm run dev
   (Opens at http://localhost:5173)

6. TEST FUNCTIONALITY
   - Connect wallet
   - Mint NFT
   - Test DeFi features
   - Verify transactions in explorer
